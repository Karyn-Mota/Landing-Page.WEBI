## Objetivo do Projeto
O objetivo deste projeto é **criar uma landing page** que apresente informações sobre um **servidor do IFCE Campus Maranguape** — podendo ser um professor, técnico administrativo, intérprete de Libras, entre outros.  
Cada dupla escolheu um servidor diferente, evitando repetições.  

A página deve demonstrar a aplicação prática dos conceitos de **estruturação (HTML)** e **estilização (CSS)**, explorando aspectos como:
- Layout responsivo  
- Tipografia e cores harmônicas  
- Uso de imagens e ícones  
- Navegação intuitiva e design limpo
